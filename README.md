# 🐍 Nova Snake

A modern and stylish Snake Game built with smooth gameplay, vibrant visuals, and exciting challenges. Control your snake, collect energy cubes, grow longer, and achieve the highest score possible!

---

## 🎮 Features

- Smooth snake movement
- Endless gameplay mode
- Score tracking system
- Increasing difficulty and speed
- Modern neon-style UI
- Sound effects and animations
- Restart and pause system
- Mobile + Desktop support

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/nova-snake.git
```

### Open in Unity

1. Open Unity Hub
2. Click Open Project
3. Select the cloned folder
4. Press Play ▶️
